## 🚀Getting Started

먼저, npm과 node 버전을 확인하세요.
```
npm @8.6.0
node @v16.15.0
```
run the development server:
```
npm run dev
# or
yarn dev
```
Open http://localhost:3000 with your browser to see the result.
